# User Interface Examples

> **Last Updated:** 2026-01-17 21:41 UTC
\nExamples for building custom interfaces with pyMediaManager's UI components.

## Overview

pyMM uses PySide6 with QFluentWidgets for a modern Fluent Design interface.

## Basic Examples

**`simple_dialog.py`** - Create a simple dialog
**`show_notification.py`** - Display notifications
**`custom_widget.py`** - Build custom widgets

## Intermediate Examples

**`custom_wizard_page.py`** - Add wizard pages
**`themed_components.py`** - Use theme colors
**`signal_handling.py`** - Handle Qt signals

## Advanced Examples

**`custom_view.py`** - Create custom main views
**`complex_layout.py`** - Build complex layouts
**`main_window_integration.py`** - Integrate with main window

## Key Concepts

- **PySide6**: Qt6 for Python
- **QFluentWidgets**: Fluent Design components
- **Signals/Slots**: Qt event system
- **Theme Support**: Light/dark mode theming
